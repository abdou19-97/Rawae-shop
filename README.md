# Rawae Cosmatics E-Commerce Website

A modern, multilingual cosmetics e-commerce platform with admin panel.

## Features

- 🌍 Multi-language support (English, Arabic, French)
- 🛒 Shopping cart with tiered pricing
- 🎁 Free gifts with products
- 💳 Multiple payment methods
- 📱 Fully responsive design
- 🔐 Admin panel for product management
- 💾 LocalStorage-based data persistence

## Quick Start

### Development

```bash
npm install
npm run dev
```

Visit: `http://localhost:5173`

### Admin Access

URL: `http://localhost:5173/admin`
Password: `rawae2026`

⚠️ **Change the password before deploying!**

### Build for Production

```bash
npm run build
```

### Deploy to Vercel

1. Push code to GitHub
2. Import project on Vercel
3. Deploy automatically

Or use Vercel CLI:

```bash
npm i -g vercel
vercel
```

## Admin Panel

- Add/Edit/Delete products
- Manage categories and subcategories
- Set discounts and promotions
- Export/Import product data
- Track inventory and sales

## Customer Features

- Browse products by category
- Filter and search
- Add to cart with bulk discounts
- Apply promo codes (RAWA10, RAWA20)
- WhatsApp order confirmation
- Wishlist functionality

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- LocalStorage for data
